📖 이용안내

RP 계산기 사용법

  1. 보유 RP를 입력합니다.
  2. 계산 버튼을 누릅니다.
  3. 남는 RP가 가장 적은 추천 조합을 확인합니다.
  4. 필요하면 포함/제외 제품과 우선 기준을 조정합니다.

희석 계산기 사용법

  1. 전체 용량을 ml 단위로 입력합니다.
  2. 원하는 희석률을 선택하거나 직접 입력합니다.
  3. 1ml 기준 방울 수를 확인합니다.
  4. 에센셜오일 방울 수와 베이스 용량을 확인합니다.

화장품 원가·판매가 계산기 사용법

  1. 원료명, 구입량, 구입가, 사용량을 입력합니다.
  2. 용기값, 라벨값, 포장비 등 부자재 비용을 입력합니다.
  3. 완성 수량과 목표 마진율을 입력합니다.
  4. 재료 원가, 총 원가, 개당 원가, 권장 판매가를 확인합니다.

주의사항

본 계산기는 사용 편의를 위한 참고용 도구입니다. 실제 제품 가격, RP, 재고, 법적 요건은 공식 정보와 관련 규정을 기준으로 확인해주세요.
